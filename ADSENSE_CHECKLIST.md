# AdSense Application Checklist — Tool Acadmy

Before applying to Google AdSense, verify all items below:

## Content Requirements
- [ ] Site has 20+ pages of original content
- [ ] 5 blog posts published (1500+ words each)
- [ ] 50+ unique tool pages indexed by Google
- [ ] Content is in English
- [ ] No copyrighted content
- [ ] No adult content

## Required Pages
- [ ] Privacy Policy page exists with cookie policy (ID: 11)
- [ ] Terms and Conditions page exists (ID: 12)
- [ ] About page exists with E-E-A-T signals (ID: 9)
- [ ] Contact page exists with working form (ID: 10)
- [ ] HTML Sitemap page created (slug: sitemap-page)

## Technical Requirements
- [ ] Site is live on real domain (toolacadmy.com)
- [ ] SSL certificate installed (HTTPS)
- [ ] Site loads fast (under 3 seconds) — check PageSpeed Insights
- [ ] Navigation is clear and functional
- [ ] All links work (no broken links) — run Screaming Frog
- [ ] Site has been live for at least 2–4 weeks
- [ ] robots.txt allows AdSense bots
- [ ] Sitemap submitted to Google Search Console

## SEO/Indexing
- [ ] Google Search Console verified
- [ ] Sitemap submitted: https://toolacadmy.com/sitemap_index.xml
- [ ] RankMath SEO plugin installed and active
- [ ] All 150+ tool pages have meta titles and descriptions
- [ ] All 150+ tool pages have JSON-LD schema

## AdSense Ad Code Preparation
- [ ] ADSENSE_PUBLISHER_ID constant defined in wp-config.php
- [ ] Ad slots placed in correct positions:
  - Homepage: leaderboard below hero, responsive between sections
  - Tool pages: leaderboard above tool, rectangle sidebar, responsive after FAQ
  - Blog posts: leaderboard below title, responsive mid-content
- [ ] Content-to-ad ratio: minimum 70% content, maximum 30% ads
- [ ] No more than 3 ads per page (AdSense policy)

## After Approval
- [ ] Add ca-pub-XXXXXXXXXX to wp-config.php as ADSENSE_PUBLISHER_ID
- [ ] Replace AUTO in ad slots with actual slot IDs from AdSense dashboard
- [ ] Enable Auto Ads as supplemental option
- [ ] Monitor performance in AdSense dashboard

## Notes
- Apply only after the site has been live on toolacadmy.com for 2+ weeks
- Make sure Google can crawl and index at least 50 pages first
- Privacy Policy MUST mention Google AdSense cookies and Analytics
- All 150+ tool pages count toward page count for AdSense review
